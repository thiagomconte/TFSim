Adicionada entrada de arquivos para valores da latência. Entre em opçoes>tempos de latência, no menu de entrada para os valores aperte cancelar e será exibida outra tela para entrada de arquivo de texto.
O arquivo de texto deve conter os números, informando um número por linha. 
No terminal será exibido o desempenho de CPI das instruções executadas, após todas as intruções serem completadas, estará exibido o CPI calculado.
